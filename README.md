# Formedix Technical Test

## What you will need

- npm `version 6.14.0`
- A Text Editor

## To Run This Website

- Open Project in Visual Code or any other text editor
- Run `npm i`
- Run `npm run dev`
- Go to relevant `localhost server`

## Libraries used

- `babel` - for packaging the project
- `eslint` - to get warnings if doing wrong code
- `prettier` - to ensure that code is properly formatted
- `react-grid-gallery` - The gallery for displaying images
- `react-loader-spinner` - The spinner for the loading of images
